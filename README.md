# 関東IT健保フォーム自動入力 Chrome拡張機能

この拡張機能は、関東IT健保の施設抽選申込フォームに保存した情報を自動入力するためのChrome拡張機能です。

## 機能

- 個人情報（記号、番号、事業所名など）を保存
- 宿泊情報（宿泊人数、希望部屋数）を保存
- ワンクリックでフォームに自動入力
- 保存したデータはブラウザのローカルストレージに保存（同期可能）
- モダンなUI（PureCSS採用）
- 自動入力成功時に完了メッセージを表示

## インストール方法

1. このリポジトリをクローンまたはダウンロードします
2. Chromeブラウザで `chrome://extensions/` にアクセスします
3. 右上の「デベロッパーモード」をオンにします
4. 「パッケージ化されていない拡張機能を読み込む」をクリックします
5. ダウンロードしたフォルダを選択します

## 使い方

1. 拡張機能をインストール後、Chromeツールバーのアイコンをクリックします
2. 「設定」タブで必要な情報を入力し、「保存」ボタンをクリックします
   - 個人情報（記号、番号、事業所名など）
   - 宿泊情報（宿泊人数、希望部屋数）
3. 関東IT健保の施設抽選申込フォームページにアクセスします
4. 拡張機能のアイコンをクリックし、「自動入力」タブの「自動入力」ボタンをクリックします
5. 保存した情報がフォームに自動入力されます
   - 部屋タイプは自動入力されません（手動で選択してください）
6. 自動入力が成功すると完了メッセージが表示されます

## セキュリティについて

- 保存されたデータはChrome Storageに保存され、ブラウザ内に留まります
- データはサーバーに送信されることはありません
- 拡張機能のアクセス許可は必要最小限に設定されています

## 技術的特徴

- PureCSSを使用したモダンなUI（Chrome拡張機能の制約に対応）
- シンプルで使いやすいインターフェース
- レスポンシブデザイン
- Chrome Storage APIによるデータ保存

## 注意事項

- この拡張機能は非公式であり、関東IT健保とは一切関係ありません
- 自己責任でご利用ください
- フォームの仕様変更があった場合、正常に動作しなくなる可能性があります
