# ITS健保フォーム自動入力

関東ITソフトウェア健康保険組合（ITS健保）の施設抽選申込フォームに、保存した個人情報をワンタップで自動入力するツールです。

PC・iPhone・Android のどのブラウザからでも利用できます。

## 使い方

### 1. ブックマークレットを作成する

[ブックマークレット生成ページ](https://taizo-pro.github.io/itkenpo-automation/) にアクセスし、必要な情報を入力して「ブックマークレットを生成」をクリックします。

### 2. ブックマークに登録する

#### PC（Chrome / Edge / Firefox）

1. 「コピー」ボタンをクリック
2. ブックマークバーを右クリック →「ページを追加」
3. 名前を「ITS健保自動入力」に変更
4. URL欄にコピーしたコードを貼り付けて保存

#### iPhone（Safari）

1. 「コピー」ボタンをタップ
2. Safariで適当なページをブックマーク
3. ブックマーク一覧を開き、保存したブックマークを「編集」
4. URL欄を全選択して削除し、コピーしたコードを貼り付け
5. 名前を「ITS健保自動入力」などに変更して保存

#### Android（Chrome）

1. 「コピー」ボタンをタップ
2. Chromeで適当なページをブックマーク
3. ブックマーク一覧を開き、保存したブックマークの「︙」→「編集」
4. URL欄を全選択して削除し、コピーしたコードを貼り付け
5. 名前を「ITS健保自動入力」に変更して保存

### 3. フォームで実行する

ITS健保の施設抽選申込フォームページを開き、登録したブックマークをタップ（クリック）すると、保存した情報が自動入力されます。

## セキュリティについて

- 入力された情報はすべてお使いのブラウザ内でのみ処理されます
- 外部サーバーへのデータ送信は一切行いません
- ソースコードはこのリポジトリで公開しています

## 注意事項

- このツールは非公式であり、関東ITソフトウェア健康保険組合とは一切関係ありません
- 自己責任でご利用ください
- 対象サイトのフォーム仕様が変更された場合、正常に動作しなくなる可能性があります
